## Building an image from a existing image

```
docker build --tag nameoftheimage -<<EOF
FROM dockerhubname
RUN the command to be executed
EOF
```

`docker run -it --rm my-ubuntu-image`
`-rm` : means to remove after exiting from the container
`-it`: `-i`for interactive `-t`for tty

## Using docker volumes

1. creating the docker volume
`docker volume create my_volume`

2. now to mount it in the docker container
`docker run -it --rm --mount source=my_volume,destination=path/tosave/docker/file myimage`

`--mount`: to mount `source=`:name of volume `destination=`: files to be saved in the volume

even if the container is permanently removed still the `volume` keeps the data saved
